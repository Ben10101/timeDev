┌────────────────────────────────────────────────────────────────────────────┐
│                       📈 FLUXO DE DADOS COMPLETO                          │
├────────────────────────────────────────────────────────────────────────────┤
│
│  1. Usuário digita ideia no Frontend React
│                          ↓
│  2. Frontend faz POST /api/generate-project
│                          ↓
│  3. Backend Node.js recebe requisição
│                          ↓
│  4. Backend chama orchestrator.py
│                          ↓
│  5. Orchestrator executa pipeline de agentes
│     • PM Agent  → Backlog
│     • RA Agent  → Requisitos
│     • Architect → Arquitetura
│     • Dev Agent → Código
│     • QA Agent  → Testes
│                          ↓
│  6. Artefatos salvos em outputs/projects/[id]/
│                          ↓
│  7. Orchestrator retorna JSON ao Backend
│                          ↓
│  8. Backend retorna JSON ao Frontend
│                          ↓
│  9. Frontend exibe resultados em 5 abas
│                          ↓
│  10. Usuário pode baixar cada artefato
│
└────────────────────────────────────────────────────────────────────────────┘
